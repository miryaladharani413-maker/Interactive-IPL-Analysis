# 🏏 Interactive IPL Analytics Dashboard

## 📌 Project Overview
Interactive IPL Analytics Dashboard is a web-based application that provides insights into team and player performance using data visualization and machine learning techniques.

---

## 🎯 Features
- 📊 Team performance analysis (wins, losses)
- 🧑‍💻 Player statistics (runs, matches, strike rate)
- 🤖 Match winner prediction using Machine Learning
- 📈 Interactive charts and visualizations
- 🎛️ User-friendly dashboard interface

---

## 🛠️ Technologies Used
- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn
